# Tony Xing — Portfolio

Personal portfolio site, self-hosted on Next.js (App Router) + React + Tailwind v4. Migrated off Framer.

## Develop

```bash
npm run dev   # http://localhost:3000
```

## Working rules

All design, content, routing, and asset decisions follow the docs in the [`rules/`](./rules) folder:

- **[`rules/PORTFOLIO_WORKING_RULES.md`](./rules/PORTFOLIO_WORKING_RULES.md)** — binding rules + process spec. Read it before changing the portfolio.
- **[`rules/DESIGN_SYSTEM.md`](./rules/DESIGN_SYSTEM.md)** — token reference (color, type, spacing, motion, components).

The agent entry point is `CLAUDE.md` → `AGENTS.md`, which both point back to that rules doc.
